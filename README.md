# kawaii_peg
ペグリンっぽいかわいいゲーム

`index.html` にアクセスするとゲームが表示されます。

## 追加要素

- ボールが下に落ちた瞬間にプレイヤーがダメージを受けるようになったよ。
- 落下時やペグヒット時にキラキラや画面シェイクなどのエフェクトがいっぱい出るよ。
- ボールは3発まで一度に投げられて、使い切ったらリロードが必要だよ。リロードにも1ターンかかって、その間も敵からダメージを受けちゃうの。
